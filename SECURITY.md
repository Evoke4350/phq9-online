# Security Policy

## Reporting a vulnerability

Email **contact@phq9.online** with a description of the issue and steps to reproduce.

Please do **not** open a public GitHub issue for security reports.

## Response expectations

- Acknowledgment within 14 days.
- A reasonable-effort fix or mitigation once the issue is confirmed.
- Credit in the changelog if you wish; anonymous reports are also welcome.

This project has no formal SLA. There is no bug bounty.

## Scope

In scope:

- Code in this repository
- Production deployment at `phq9.online`

Out of scope:

- Third-party services (Google AdSense, Fly.io platform, GitHub)
- Issues that require physical access to a user's device
- Best-practice or hardening suggestions that are not exploitable

## PGP

PGP key available on request.
